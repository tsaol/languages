Initial release.

- 3 decks: daily (88), express (659), vocab (147)
- Parse ~/english.log into categorized error entries
- Notion vocabulary sync (147 curated work words)
- Notion progress sync for cross-device support
- Conversation practice mode with 20 work scenarios
- Interactive quiz with translate/correct modes
- Simplified 3-level rating (Don't know / Hesitated / Easy)
- Flask web UI on port 5555 (dark theme, mobile-first)
- CLI with shortcuts (elv, ele, elr, elt, elq, eld)
- Cron job for daily auto-sync at 6am
- Progress dashboard and statistics
